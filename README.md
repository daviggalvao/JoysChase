# Joy's Chase
Remake of **Pac-Man** inspired in "**Inside Out**"

## How to play
Drag the file `JoysChase.s` over the executable [FPGRARS](https://github.com/LeoRiether/FPGRARS) (included in this repository)
## Controls
```
[W] [A] [S] [D] - Move Joy
[0] - Close game
[1] - Start level 1
[2] - Start level 2
```
